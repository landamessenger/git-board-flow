# Git Board Flow

**Git Board Flow** is a powerful GitHub Action designed to streamline issue, branch, and project management while adhering to the Git-Flow methodology. This action simplifies your workflow by automating the creation and management of branches, linking them to issues, and ensuring smooth integration with GitHub Projects.

## Key Features

Git Board Flow offers several powerful features to streamline your GitHub workflow:

1. **Automatic Branch Creation** - Creates and links branches based on issue labels
2. **Commit Monitoring** - Tracks commits and updates issues automatically
3. **Pull Request Linking** - Connects PRs to corresponding issues
4. **GitHub Project Integration** - Seamlessly manages projects and tasks
5. **Efficient Execution** - Completes tasks in under 20 seconds

## Quick Start

To get started with Git Board Flow, add the following workflow to your repository:

```yaml
name: Git Board Flow - Issue

on:
  issues:
    types: [opened, edited, labeled, unlabeled]

jobs:
  git-board-flow:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: landamessenger/git-board-flow@master
        with:
          github-token: ${{ secrets.GITHUB_TOKEN }}
          github-token-personal: ${{ secrets.REPO_PAT }}
```
